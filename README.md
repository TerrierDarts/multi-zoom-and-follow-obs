# multi-zoom-and-follow-obs
Script for OBS and Multi Level Zooms and Mouse Follow

## OBSRaw Hotkeys (WEBSOCKET 5)

### Zoom Off

```json
{
  "requestType": "TriggerHotkeyByName",
  "requestData": {
    "hotkeyName": "zoom_off"
  }
}
```

### Zoom 2x

```json
{
  "requestType": "TriggerHotkeyByName",
  "requestData": {
    "hotkeyName": "zoom_two"
  }
}
```

### Zoom 4x

```json
{
  "requestType": "TriggerHotkeyByName",
  "requestData": {
    "hotkeyName": "zoom_four"
  }
}
```
### Follow Toggle

```json
{
  "requestType": "TriggerHotkeyByName",
  "requestData": {
    "hotkeyName": "follow_toggle"
  }
}
```
